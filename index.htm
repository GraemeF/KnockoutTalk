What is KnockoutJS?

  • JavaScript library (39KB, no dependencies) by Steve Sanderson
  • Works on most browsers —
    IE 6+, Firefox 2+, Chrome, Opera, Safari (desktop/mobile)
  • MVVM for your web browser
  • Bind UI elements to your models

What's MVVM?

  • Model
    Your domain object
  • View
    UI element that presents your model
  • ViewModel
    Maps between Model and View

MVVM in Knockout

  • Model
    Server-side domain objects
  • View
    HTML elements that display your model in the page
  • ViewModel
    JavaScript object exposing bindable properties and communicating with server via AJAX

Why care?

By encapsulating data and behavior into a view model, you get a clean, extensible foundation on which to build sophisticated UIs without getting lost in a tangle of event handlers and manual DOM updates.

View Models

View Models

  • Any JavaScript object will do:
    var vm = { name: "Dougal", occupation: "Priest" };
  • Bind to properties:
    <span data-bind="text: name"></span>
  • Tell Knockout to bind your DOM elements to the ViewModel:
    ko.applyBindings(vm);
  • Result:
    <span>Dougal</span>

What's wrong with that?

var vm = { name: "Dougal", occupation: "Priest" };
            ko.applyBindings(vm);
  • Change the name:
    vm.name = "Ted";
  • Result:
    <span>Dougal</span>

Observables

Observables

var vm = { name: new ko.observable("Dougal"),
                            occupation: "Priest" };
                 ko.applyBindings(vm);
  • View: <span>Dougal</span>
  • Set the name observable: vm.name("Ted");
  • View changes to: <span>Ted</span>

Demo

Observables

  • Create with an initial value:
    var foo = ko.observable(123);
  • Change the value:
    foo(456);
  • Get the current value:
    foo(); // returns 456

Observable Arrays

var vm = {
                   priests: new ko.observableArray(["Dougal", "Ted"]) };
                 ko.applyBindings(vm);
  • Replace: vm.priests(["Dick", "Noel"]);
  • Supports array functions: pop, push, shift, unshift, reverse, sort, splice

Computed Observables

var ViewModel = function () {
              var self = this;
              this.numbers = ko.observableArray([1, 2, 3]);
              this.words = ko.observable("Total: ");
              this.total = ko.computed(function () {
                var sum = 0;
                ko.utils.arrayForEach(self.numbers(), function (x) {
                  sum += x;
                });
                return sum;
              });
              this.message = ko.computed(function () {
                return self.words() + self.total();
              });
            };
